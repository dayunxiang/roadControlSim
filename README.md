# simulation
optimization and simulation system

# 安装
* cd simulation
* npm install || yard add
* npm run dev

# 打包-发布版本
* npm run build

# 代码规范
* 格式要求请遵循Eslint规范
* 新建文件请参照现有目录containers下文件的创建与开发


# 文件模块
* Header 头部文件
* Nav 时间
* Login 登录
* Entrance 入口 选择功能模块

* InterPlan 路口组织交通方案设计
    - Evaluate 评价
    - Facilitiesmana 设施
    - Flow 流量
    - Informationmana 信息管理
    - Navigation 导航
    - Projectmana 项目管理
    - Roadtraffic 道路交通

* TrafficAnalysis 交通预测分析
* TrafficDatas 数据分析
* TrafficManage 仿真资源管理
* TrafficArea 区域交通组织方案设计
    - TrafficArea 方案设计
    - ProManage 方案管理
    - InfoManage 信息管理
* TrafficSystem 系统管理
